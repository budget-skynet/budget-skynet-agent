## ❌ НЕ МОГУ — требует ручных действий или недостижимо автоматически

### 🎯 Непредсказуемый результат — качество не гарантируем
- **Poe Bot** — деплой возможен, но публикация в Directory ручная → скип
- **Gradio с GPU** — не гарантируем запуск на GPU окружении
- **Парсеры / скрейперы** — если сайт защищён Cloudflare/CAPTCHA → результат не гарантирован
- **Telegram Bot с платежами** — платёжная часть не тестируется
- **Data science с приватным датасетом** — нет доступа к данным заказчика
- **Аналитика с метриками** ("400+ downloads", "1000+ users") → скип

### 🏪 Платформы без API / с ручной модерацией
- **GPT Store (OpenAI)** — публикация только через UI chat.openai.com, нет API
- **JetBrains Plugin Marketplace** — требует Kotlin/Java + ручная проверка командой JetBrains
- **Chrome Web Store** — нужен аккаунт разработчика + ручная публикация + ревью
- **Firefox Add-ons (AMO)** — аналогично Chrome
- **Apple App Store / Google Play** — нет API для публикации
- **VS Code Marketplace** — можно генерировать код, но публикация через `vsce` требует токен разработчика
- **Poe Bot Directory** — листинг в каталоге только вручную через форму

### 📊 Задачи с гарантией метрик
- **"500+ загрузок/скачиваний"** — невозможно гарантировать
- **"100+ инсталляций"** — невозможно гарантировать  
- **"1000+ пользователей"** — невозможно гарантировать
- **"Viral reach / trending"** — невозможно гарантировать

### 📱 Социальные сети (требует живой аккаунт)
- **Twitter/X посты** — нет аккаунта, нельзя постить
- **LinkedIn посты** — нет аккаунта
- **Reddit посты** — нет аккаунта (только генерация текста)
- **Discord посты в чужие серверы** — нет доступа
- **"Link to your tweet/post"** — невозможно, нет аккаунтов
- **Influencer outreach** — требует живого человека

### ⛓️ Сложный Web3 / Blockchain
- **Solidity смарт-контракты** — нет компилятора и деплоя в E2B
- **Rust смарт-контракты NEAR** — компиляция требует специального окружения
- **Security Audit** — требует специализированных инструментов и экспертизы
- **MEV / Arbitrage / Sniper боты** — запрещено политикой

### 🎨 Дизайн и визуальный контент
- **Figma / UI дизайн** — нет доступа к Figma API
- **Видео** — монтаж, запись экрана, анимация невозможны
- **Изображения / Инфографика** — нет image generation API
- **3D модели** — не поддерживается

### 🏗️ Сложная инфраструктура
- **Kubernetes / Terraform** — генерация конфигов возможна, но деплой нереален
- **Деплой на AWS/GCP/Azure** — нет доступа к облачным аккаунтам
- **Docker publish** — нет доступа к Docker Hub аккаунту заказчика
- **Database setup** — создание и настройка БД требует сервера

### 🎰 Финансовые риски
- **Trading ботов** — автоматическая торговля на биржах
- **High-frequency trading** — не поддерживается
- **Алгоритмический трейдинг** — нет доступа к биржевым API заказчика

## 📌 Важные заметки

- **Gist** — основной способ сдачи работы. Все файлы публикуются в публичный GitHub Gist
- **E2B Sandbox** — для тестирования pip/npm пакетов перед сдачей (Linux окружение, 60 сек)
- **Модель выбора:** Haiku для большинства задач, Sonnet автоматически для конкурсов 100+ NEAR
- **Цена бида:** 75% от бюджета для задач >2 NEAR, 100% для мелких
- **Если что-то пошло не так** при выполнении — уведомляю владельца в Telegram, НЕ сдаю сырую работу заказчику
